# **App Name**: DiffSense

## Core Features:

- Pull Request Webhook Integration: Automatically listens for open or updated pull requests from GitHub or GitLab to initiate scans.
- Automated Context Analyzer: A reasoning tool that processes the codebase structure and diff history to provide relevant analysis for every change.
- Smart Logic & Security Guard: Generative AI logic to identify security vulnerabilities, memory leaks, and performance bottlenecks within the modified code.
- Actionable Inline Reviewer: Automatically posts precisely located, markdown-formatted comments directly to the specific lines in the PR.
- Engineering Health Dashboard: Visual interface displaying PR health scores, frequent hotspots of technical debt, and review statuses.
- Review Standards Configurator: Set specific project guidelines for the AI to prioritize, such as accessibility standards or strict typing.

## Style Guidelines:

- Primary color: Precision Indigo (#6489E7) symbolizing intelligence and clarity.
- Background color: Deep Slate (#16171D) providing a high-contrast dark theme suitable for late-night engineering sessions.
- Accent color: Neon Cyan (#2AA9EE) to highlight active bugs and calls to action.
- Headline font: 'Space Grotesk' for a computerized, technical feel. Body font: 'Inter' for maximum readability during complex code reviews.
- Code font: 'Source Code Pro' used for all code diffs and logic snippets.
- A structured, bento-box grid layout to separate repository health metrics from detailed PR lists.
- Micro-transitions including logic-flow pulses during active scanning and smooth status shifts when a PR is cleared.
